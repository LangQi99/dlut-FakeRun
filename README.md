# dlut-FakeRun

内置了dlut的操场路径 可以模拟校园跑 随时更改速度

> [!CAUTION]
> 本项目仅供娱乐，请合理使用。


## 使用方式

1. 启用`开发者选项`
2. 在`选择位置模拟信息应用`中选择该应用

- Android 11以下环境未经测试
- 真机环境目前暂时不支持后台运行 建议前台运行或者悬浮窗
- 真机环境技术上不支持 `Keep` 但支持 `小米运动健康` `华为运动健康` 等
- 模拟器环境下支持 `Keep` 例如PC端的 `BlueStacks 5 China`
